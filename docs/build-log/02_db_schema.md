## Notable

- **Mid-chunk closeout required a Postgres instance.** Railway Postgres was provisioned via CLI (project `wiredhowse-magic-link`, US West, `zephyr.proxy.rlwy.net:40212`). This is the first Railway resource for the project.
- **FK naming reconciliation:** The hand-written migration originally used shorter FK constraint names that wouldn't match what drizzle-kit's snapshot expected. Resolution: replaced `0000_initial.sql` with drizzle-kit's canonical output, plus a manually added `CREATE EXTENSION IF NOT EXISTS "citext"` line (drizzle-kit doesn't emit extension DDL). This is the kind of subtle issue that would have bitten three weeks later if not caught now.
- **Pattern locked in for future schema changes:** edit `schema.ts` → run `pnpm db:generate` → commit the new SQL + updated snapshot.
